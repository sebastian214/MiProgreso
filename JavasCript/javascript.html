Interpolacion de variables

La interpolacion de varibles es meter dentro de una cadena de texto el valor dinamicamente de una variable

Para realizar esta interpolacion tenemos que hacer uso de una caracteristica de las cadenas de texto llamada Template String

Para esta interpolacion utilizamos el agento grave o invertido ``

Otra caracteristica de la comilla invertida

Las comillas invertidas tambien se utilizan para generar contenido html automaticamente ya que permite colocar cadenas de texto de forma desordenada con saltos de linea y indentaciones html

De esta manera podemos no solo concatenar, tambien podemos utilizarlo para otras propiedades
