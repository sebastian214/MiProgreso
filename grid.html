Posicionamientos con Grid Lines


Para mover las grid celldas entre las lineas utilizamos grid-row-star, grid-row-end / grid-column-start, grid-column-end asignaod desde que linea comienza hasta donde termina en los dos ejes o bien podemos ustar los atajos: grid-row y grid column colocando el inicio y el final con barra invertida de por medio. Ejemplo sintaxis: grid-row: 3/4; el 3 va a especificar el inicio y el 4 la linea final


Posicionamiento con Grid-areas grid-areas va a funcionar como un shorthand de grid-row y grid-column separados por la barra invertida. sintaxis: grid-area: 2/4 3/6;
grid-area: row start / column start / row end /column end

El valor Span: utilizamos span para definir cuanto abarca en lineas una celda

grid-template-columns: [linea-c1] 1fr [linea-c2] 1fr [linea-c3] 1fr; En este ejemplo asignamos tres lineas con sus respectivos nombres y proporcion, luego lo hacemos con las filas igualmente
